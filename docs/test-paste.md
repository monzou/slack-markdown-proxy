**Slack Markdown Proxy** converts your Markdown on paste.

Supports *italic*, **bold**, and [links](https://github.com).

Key features:
- Automatic paste detection
- **Nested** formatting like *[italic links](https://example.com)*
  - Bullet lists with indentation
  - Mixed **bold** and *italic*
- Blockquotes and lists

1. Paste Markdown into Slack
2. Get rich text instantly
3. No configuration needed

> Markdown should just work in Slack.
